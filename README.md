## üöÄ Getting Started

### Prerequisites
- Node.js installed
- MongoDB running locally (default: `mongodb://localhost:27017/music_mood`)

### Installation

1. **Clone the repository** (if applicable) or navigate to the project folder.

2. **Setup Backend**:
   ```bash
   cd server
   npm install
   ```

3. **Setup Frontend**:
   ```bash
   cd client
   npm install
   ```

### Running the App

1. **Start the Server**:
   ```bash
   cd server
   npm run dev
   ```
   The backend will run on `http://localhost:5000`.

2. **Start the Client**:
   ```bash
   cd client
   npm run dev
   ```
   The frontend will run on `http://localhost:5173`.
